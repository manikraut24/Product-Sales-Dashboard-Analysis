# Product-Sales-Dashboard-Analysis 📊
This project analyzes a product dataset containing categories such as Furniture, Technology, and Office Supplies.  The goal is to explore product distribution across categories and build an interactive dashboard for better insights.

## Dataset

The dataset contains the following columns:

- Category – Main product category (Furniture, Technology, Office Supplies)
- Sub-Category – Product sub-category (Binders, Phones, Tables, etc.)
- Product Name – Name of the product

Sample Data:

| Category   | Sub-Category |         Product Name         |
|------------|--------------|------------------------------|
| Furniture  | Bookcases    | Westfield Westfield Bookcases|
| Technology | Phones       | GE 30522EE2                  |
| Office Sup.| Storage      | Recycled Steel Personal File |

## 📈 Key KPIs 

- Total Sales: 1.6M
- Total Quantity Sold: 22K
- Total Orders: 175K
- Average Shipping Days: 4 Days

## 📊 Dashboard Insights
 
Consumer segment generates the highest sales (48%)
- COD payment mode contributes the largest share (43%)
- Office Supplies category records the highest sales
- Standard Class is the most used shipping method
- Phones, Chairs, and Binders are the top-performing sub-categories
- California and New York generate the highest sales

## Forecast Analysis
A 15-day sales forecast was created using Power BI forecasting to predict future sales trends based on historical data.
This analysis helps businesses in:
- Inventory planning
- Logistics optimization
- Strategic decision making)
